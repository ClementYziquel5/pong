Notes pour l'utilisation:
  - Ouvrir headers/settings.h pour personnaliser vos paramètres de jeu (Paramètres par défaut conseillés)
    make
    ./Pong

Commandes:
  - Lancer la balle        : B
  - Redémarrer la partie   : R
  - Mettre le jeu en pause : Barre espace

  - Joueur de gauche:
            - Monter       : E
            - Descendre    : D

  - Joueur de gauche:
            - Monter       : O
            - Descendre    : L
